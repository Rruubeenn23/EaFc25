Una vez que ejecutas el programa, lo que te va a salir es un menu de distintas opciones. 
## Opciones: 

1. Abrir Sobres
2. Ver Club
3. Hacer Plantilla
4. Ver Plantilla
5. Jugar Partido
6. Salir

Cada uno de estas opciones tiene sus propias funcionalidades. Aquí va la explicación de cada una.
